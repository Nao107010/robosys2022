# インストール手順
コマンドを使用するには、リポジトリをクローンしたあと、robosys2022へ移動してください。

```
$ git clone https://github.com/Nao107010/robosys2022.com
$ cd robosys2022
```

また、Pythonを用いたプログラムなので未インストールの方はインストールをお願いします。

# plusコマンド

標準入力から読み込んだ数字を足し合わせて出力します。

## 実行結果例
```
$ seq 10 | ./plus
55.0
```

# 動作確認済み環境
次の環境で動作確認がされています。
* Ubuntu 18.04
* Python 3.7～3.10
